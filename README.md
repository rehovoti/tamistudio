# Tami Studio (Static)

Mobile-first marketing page for a small business specializing in brow design and lash lifting.

## Run locally

Option A (quick): open `index.html` in your browser.

Option B (recommended): serve locally (prevents some browser restrictions).

```bash
cd /Users/emilfeldman/proj/TamiLashes/TamiStudio
python3 -m http.server 5173
```

Then open:

- http://localhost:5173

## Customize

- Replace the placeholder WhatsApp/phone/address in the Contact section.
- Swap the Unsplash placeholder images with real studio/client lifestyle photos (with permission).
- Edit the text in `app.js` under `translations.he` and `translations.en`.
